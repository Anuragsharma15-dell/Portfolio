# React Portfolio - Anurag Sharma

This is a single-page **React portfolio website** built with **TailwindCSS** and **Framer Motion**, designed to showcase my skills, projects, and background with smooth animations and elegant UI. 

## 🚀 Features
- Modern UI built using **React + TailwindCSS**
- Smooth transitions & animations using **Framer Motion**
- **About Me** section introducing my education and passion
- Interactive **Projects** section with animated cards
- **Contact section** with social links and email option
- Fully responsive layout
